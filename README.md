# dietary-plan-server
基于SpringBoot+Vue前后端分离的健康饮食网系统（源码+文档），系统包含管理员、用户角色，功能如下： 管理员：首页统计（帖子统计、注册人数统计）、用户管理、菜谱管理（菜谱食材管理）、菜谱评价、食材管理、论坛管理（版块管理、帖子管理、评论管理）、饮食知识、BMI指标、平台留言、公告信息、个人信息密码修改。 用户：注册、登录、首页、菜谱推荐（菜谱详情、收藏、食材、评价）、社区互动（版块、帖子列表、帖子详情、点赞、评论）、饮食知识、平台留言、系统公告。用户还具有个人中心功能，功能包括：饮食建议、菜谱收藏、菜谱评价、我的帖子、我的点赞、帖子评论、平台留言、个人信息、密码修改。
